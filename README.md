# clash-royale-player-stats
A aplicação permite aceder a toda a informação relativa a um jogador de Clash Royale.
É possivel fazer uma pesquisa atravez da tag do jogador que pode ser consultada no
proprio jogo. Apos a pesquisa irá ser mostrado informação como: Nome do jogador, cla,
trofeus(ranking), ciclo dos baus mais importantes(tempo que falta para ganhar), nivel
do jogador, experiencia etc.
É também possivel guardar jogadores como favoritos para que nao seja necessario estar
sempre a introduzir a tag para aceder a informação pretendida. Estes jogadores gravados
serão mostrados numa lista e com um simples clique é possivel aceder ao perfil de cada
um.
